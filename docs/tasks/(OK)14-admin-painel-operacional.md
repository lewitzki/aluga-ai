# 14 - Admin painel operacional

## Objetivo
Montar visao operacional do admin para gestao diaria de ferramentas e emprestimos.

## Escopo tecnico
- Criar pagina admin com cards/listas de status operacional.
- Exibir ferramentas cadastradas, disponiveis e emprestadas.
- Mostrar tempo corrido dos emprestimos ativos.

## Criterios de aceitacao
- [ ] Admin enxerga quantidade e lista de ferramentas por estado.
- [ ] Admin acompanha emprestimos ativos com tempo correndo.
- [ ] Filtros basicos por status e busca textual funcionam.
- [ ] Cliente nao acessa esse painel.

- `07-admin-crud-de-ferramentas.md`
