# 05 - Catalogo publico de ferramentas com filtros

## Objetivo
Entregar listagem publica de ferramentas disponiveis com filtros de busca.

## Escopo tecnico
- Criar endpoint/rota Inertia publica para catalogo.
- Implementar filtros por descricao, faixa de preco e disponibilidade.
- Paginar resultados e manter query string no frontend.

## Criterios de aceitacao
- [ ] Visitante acessa URL publica de catalogo sem login.
- [ ] Filtros combinados retornam resultados coerentes.
- [ ] Ferramenta indisponivel no periodo nao aparece como disponivel.
- [ ] Tela segue layout e componentes UI ja adotados.
