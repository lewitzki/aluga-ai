# Arquitetura do Projeto (Laravel + Inertia + React)

- `01-visao-geral.md`: stack, fluxo de requisicao e decisoes arquiteturais.
- `02-backend-laravel.md`: estrutura Laravel, middlewares, rotas e autenticacao.
- `03-frontend-inertia-react.md`: bootstrap Inertia, layouts, paginas e componentes.

- `05-organizacao-de-pastas.md`: convencoes de organizacao e responsabilidades por pasta.
