# Arquitetura C4 - Plataforma de Emprestimo de Ferramentas

Este diretorio documenta a arquitetura do MVP usando o modelo C4.

## Estrutura

- `01-contexto.md` (C1): visao de contexto, atores e sistemas externos.
- `02-containers.md` (C2): containers da solucao e responsabilidades.
- `03-componentes.md` (C3): principais componentes internos da aplicacao.

## Escopo do MVP

- Cadastro e autenticacao para `admin` e `cliente`.
- Catalogo publico de ferramentas com filtros.
- Fluxo de emprestimo com prazo determinado.
- Pagamento emulado/mockado (sem gateway real).
- Painel admin com operacao e visao financeira.
- Painel cliente com acompanhamento de emprestimos e historico.
